Задача I-Алгоритм Левенштейна

Полный балл:400
Штраф за посылку:1
Ограничение времени:3 с
Ограничение реального времени:3 с
Ограничение памяти:64M

Дано две строки. Нужно найти количество исправлений, необходимых для получения из первой строки вторую. Исправления бывают трех видов: вставка символа на произвольное место, удаление произвольного символа и замена произвольного символа любым другим символом.

Формат входных данных

В первой строке вводится строка A, во второй строка B. Обе строки состоят из маленьких латинских букв. Длина каждой строки не превосходит 500.

Формат выходных данных

Одно число — результат.

Примеры

Ввод Вывод 
aaaaa
aaaaa
 
5

Ввод Вывод 
abcdef
cdefab
 
4

Ввод Вывод 
abcdea
bcdeab
 
2
