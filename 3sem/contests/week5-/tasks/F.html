Задача F-Удаление коротких слов

Полный балл:200
Штраф за посылку:1
Ограничение времени:1 с
Ограничение реального времени:1 с
Ограничение памяти:64M

Распечатать введенное предложение, удалив из него короткие слова, которые состоят не более чем из 3 букв. (Пробелы удалять не требуется!)

Формат входных данных

На вход подается не пустая строка < 256 символов, состоящая из символов английского алфавита и пробелов. Слово - последовательность любых непробельных символов. Слова во входной строке разделены ровно одним пробелом.

Формат выходных данных

Строка, содержащая только длинные слова и пробелы. (Если в исходной были только короткие строки, то на выходе строка состоит только из пробелов.)

Примеры

Ввод Вывод 
Hello world
 
Hello world

Ввод Вывод 
Mary had a little lamb
 
Mary   little lamb
