Задача G-Максимальная возрастающая подпоследовательность

Полный балл:300
Штраф за посылку:1
Ограничение времени:1 с
Ограничение реального времени:1 с
Ограничение памяти:64M

Дана последовательность целых чисел. Необходимо найти длину её самой длинной строго возрастающей подпоследовательности.

Формат входных данных

Число n (0 < n < 100) — количество элементов в последовательности

Последовательность из n целых чисел

Формат выходных данных

Одно число — результат.

Примеры

Ввод Вывод 
3
-1 0 1
 
3

Ввод Вывод 
7
1 -2 2 -1 3 -1 4
 
4
